# 11. Raccourcis clavier

Appuyez sur `Cmd/Ctrl+/` dans l'app pour ouvrir la fenêtre de raccourcis cherchable. Sur macOS,
utilisez **Cmd (⌘)** ; sur Windows/Linux, **Ctrl**. « Opt » = ⌥ sur macOS / Alt ailleurs.

## Général
| Raccourci | Action |
|---|---|
| `Cmd/Ctrl+K` | Ouvrir la palette de commandes |
| `Cmd/Ctrl+P` | Recherche globale |
| `Cmd/Ctrl+/` | Afficher les raccourcis clavier (cette fenêtre) |
| `Cmd/Ctrl+,` | Ouvrir les Réglages |
| `Cmd/Ctrl+F` | Rechercher dans la session courante |
| `Échap` | Fermer le dialogue / panneau du dessus |

## Disposition & navigation
| Raccourci | Action |
|---|---|
| `Cmd/Ctrl+B` | Afficher/masquer la barre latérale |
| `Cmd/Ctrl+\` | Basculer la disposition en deux volets |
| `Cmd/Ctrl+1…9` | Aller à l'onglet 1–9 |

## Saisie du chat
| Raccourci | Action |
|---|---|
| `Entrée` | Envoyer le message |
| `Maj+Entrée` | Saut de ligne dans le composer |
| `Échap` (chat actif) | Annuler la réponse en cours |
| `@` | Mentionner un fichier / agent / connecteur |
| `/` | Palette de commandes slash |

## Panneaux & overlays
| Raccourci | Action |
|---|---|
| `Cmd/Ctrl+Maj+K` | Basculer l'overlay de recherche globale |
| `Cmd/Ctrl+Maj+S` | Bibliothèque de snippets |
| `Cmd/Ctrl+Maj+P` | Sélecteur de persona |
| `Cmd/Ctrl+Maj+T` | Lanceur de tests |
| `Cmd/Ctrl+Maj+R` | Visionneuse de trace de raisonnement |
| `Cmd/Ctrl+Maj+I` | Insights de session |
| `Cmd/Ctrl+Maj+O` | Sélecteur de reprise de session |
| `Cmd/Ctrl+Maj+F` | Vue focus |
| `Cmd/Ctrl+Maj+L` | Gestionnaire de skills |

## Multi-agent
| Raccourci | Action |
|---|---|
| `Cmd/Ctrl+Maj+M` | Lanceur d'orchestrator multi-agent |
| `Cmd/Ctrl+Maj+A` | Tableau de bord des sous-agents |

## Diagnostics & power-user
| Raccourci | Action |
|---|---|
| `Cmd/Ctrl+Maj+D` | Diagnostics de sécurité |
| `Cmd/Ctrl+Maj+/` | Question éclair ponctuelle (`/btw`) |
| `Cmd/Ctrl+Opt+S` | Arrêt d'urgence de l'agent actif (global) |
| `Cmd/Ctrl+R` | Recharger le renderer |

> **Note.** Quelques raccourcis dépendent de la plateforme et du build (certains sont des raccourcis
> OS globaux, d'autres des handlers internes). La fenêtre `Cmd/Ctrl+/` reflète toujours les
> raccourcis disponibles dans votre version.
